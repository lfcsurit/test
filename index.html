PROJECTS
INTRUSION DETECTION SYSTEM

Our team built an Intrusion Detection System that is solely based on raspberry pi which acts as the main framework for the operation. The primary aim of this project was to set up a secure network environment by creating a firewall, DHCP and DNS server entirely based on RPi.

This system detects security threats attacks as and when they happen by providing real time network monitoring and puts forward information about the attack to the user. The best way to understand how a network protocol works is by understanding the network from packet level. The easiest way for an attacker to intrude WNs is through the data link layer. The data link layer consists of two sub layers as: MAC (Medium Access Control) and LLC (Logical Link Control). The most popular wireless technology is considered to be 802.11 wireless technology. So this project is based on the idea of all the intrusion and detection based on the same wireless technology.
